# 掼蛋 (Guandan) - 在线卡牌游戏

基于 Spring Boot + Vue 3 的在线掼蛋卡牌对战游戏。

## 开发阶段
- Phase 1: 账号体系与基础页面

## 版本
- v0.1.0-alpha (Phase 1)

## 技术栈
- 后端：Java 17, Spring Boot 3.2.0, MyBatis-Plus 3.5.5, MySQL, Redis, JWT
- 前端：Vue 3, Vite, Element Plus, Axios
