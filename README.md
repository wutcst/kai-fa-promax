# 掼蛋 (Guandan) - 在线卡牌游戏

一个基于 Spring Boot + Vue 3 的在线掼蛋卡牌对战游戏。

## 技术栈

### 后端
- Java 17 + Spring Boot 3.2.0
- MyBatis-Plus 3.5.5
- WebSocket（实时对战通信）
- MySQL 8.0 + Redis
- JWT 认证
- Knife4j API 文档

### 前端
- Vue 3（Composition API）
- Vite 构建工具
- Element Plus UI 组件库
- Axios HTTP 客户端

## 快速开始

### 环境要求
- JDK 17+
- Maven 3.6+
- Node.js 18+
- MySQL 8.0+
- Redis 7+

### 后端启动
```bash
cd backend
mvn spring-boot:run
```

### 前端启动
```bash
cd frontend
npm install
npm run dev
```
