## ✨ 功能预览

<div>
  <table>
    <tr>
      <td width="50%" valign="top">
        <h3>🔐 用户认证</h3>
        <p>注册自动分配 6 位数字账号，JWT 登录 + Token 刷新 + 密码重置（邮箱验证码 + 限时 Token）。BCrypt 加密，前端表单校验，记住密码持久化。</p>
        <h3>🏠 房间管理</h3>
        <p>创建房间生成唯一 6 位房间号，加入/退出/解散/踢人/权限转移。满 4 人锁定，一键复制房间号，大厅列表实时刷新。</p>
        <h3>⚔️ 游戏对战</h3>
        <p>四人掼蛋经典玩法，WebSocket 实时同步，30s 倒计时自动过牌。AI 出牌推荐引擎 + 自学习权重 + 出牌动画。支持观战模式和游戏回放。</p>
      </td>
      <td width="50%" valign="top">
        <h3>🔄 匹配服务</h3>
        <p>快速匹配队列，满 4 人自动开房通知。取消匹配 + 60s 轮询超时提示。断线重连自动恢复房间状态和手牌。</p>
        <h3>📊 个人中心</h3>
        <p>战绩详情、胜率统计、时间/结果筛选、连胜/连败趋势。支持导出 JSON / CSV 格式。</p>
        <h3>🤖 AI 辅助</h3>
        <p>AI 出牌建议引擎 + 牌型自学习权重调整 + 规则问答。基于历史对局动态优化推荐。</p>
        <h3>🛡 安全加固</h3>
        <p>Token 限流防刷、API 速率控制（令牌桶）、Nginx 安全响应头（CSP/XSS/Clickjacking）。</p>
      </td>
    </tr>
  </table>
</div>

## 🚀 快速开始

<details open>
<summary><b>📋 环境要求</b></summary>

- JDK 17+ · Node.js 18+ · MySQL 8.0+

```bash
# 1. 克隆项目
git clone https://github.com/wutcst/kai-fa-promax.git
cd kai-fa-promax

# 2. 初始化数据库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS guandan_game DEFAULT CHARACTER SET utf8mb4;"
mysql -u root -p guandan_game < backend/src/main/resources/schema.sql

# 3. 启动后端
cd backend
mvn clean install -DskipTests && mvn spring-boot:run
# → http://localhost:8080

# 4. 启动前端（新终端）
cd frontend
npm install && npm run dev
# → http://localhost:5173
```

</details>

<details>
<summary><b>🐳 Docker 一键部署</b></summary>

```bash
docker compose up -d
```

</details>
